# Release Notes

## v0.3.0

当前版本具备以下能力：

- 完整的一期编译器基线
- 受控整数子集的 IR 与保守优化链路
- `char`
- 字符与字符串字面量
- 数组声明与索引
- 基础指针声明、取地址与解引用
- IR 控制流图输出
- 更完整的测试与示例

## 版本演进摘要

### v0.1.0

- 建立项目文档
- 初始化目录结构
- 完成基础 Git 仓库

### v0.2.0

- 完成第一代编译器基线
- 打通词法、语法、语义、IR、后端、CLI 和端到端
- 建立系统化测试集

### v0.3.0

- 加入适度优化
- 新增 IR 可视化输出
- 补充递归、作用域相关测试
- 扩展 `char`、字符串、数组、基础指针能力
- 增加公开仓库文档页

## 当前已知限制

- 还没有完整 C 类型系统
- 还没有结构体、联合体、枚举
- 还没有原生汇编后端
- 高级语言特性当前走 AST 规范化 C 后端，而不是 IR 后端
- Linux 端到端能力在当前机器上未做最终实机验收
